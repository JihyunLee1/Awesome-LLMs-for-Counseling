# Awesome LLMs for Counseling 🧠💬  
*A curated list of papers, datasets, benchmarks, and resources on Large Language Models for counseling, psychotherapy, and mental health support.*

![Awesome](https://awesome.re/badge.svg)
![Visitors](https://visitor-badge.laobi.icu/badge?page_id=JihyunLee1.Awesome-LLMs-for-Counseling)


## 📌 Scope

This repository focuses on **research-oriented and evaluation-driven work** on:

* LLMs for **mental health counseling**
* Therapeutic dialogue modeling (CBT / ACT / MI)
* Empathy, validation, and emotional alignment
* Safety, ethics, and harm-aware response generation
* Simulation-based training and evaluation
* Human–LLM alignment in therapy contexts
